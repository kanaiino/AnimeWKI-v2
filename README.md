# AnimeWKI

## Запуск разработки

1. Установите зависимости:
   - `cd frontend`
   - `npm install`
   - `cd backend`
   - `npm install`

2. Запустск бэкенда (в двух терминалах):
   - `cd backend`
   - `npm run server`
(json-server на порту 5002)
   - `cd backend`
   - `node server.js`
(kodik-proxy на порту 5003)

3. Запустиск фронтенда:
   - `cd frontend`
   - `npm run dev`
